# Python Expense Tracker

A clean and practical CLI-based Expense Tracker built using pure Python.  
Designed as a project to demonstrate backend logic, file handling, data processing, and CLI design.

---

## Features

- Add expenses (amount, category, note, timestamp)
- View expenses with clean formatted output
- Predefined categories (Food, Transport, Shopping, Bills, etc.)
- Update existing entries
- Delete entries using ID
- Monthly summary (total + category breakdown)
- Export all data to CSV
- JSON-based storage (data.json)
